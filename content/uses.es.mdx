## Editor y Terminal

### Editor

**[Cursor](https://cursor.sh)** — fork de VS Code con IA. Intenté resistirme pero el autocompletado por tabulador y las funciones de chat son genuinamente útiles para la productividad.

Configuraciones que me importan:
- Tema: **One Dark Pro**
- Fuente: **Geist Mono** a 13px, con ligaduras
- Extensiones: ESLint, Prettier, Tailwind CSS IntelliSense, GitLens, Error Lens

### Terminal

**[Warp](https://warp.dev)** como mi terminal principal — el autocompletado y la búsqueda de comandos con IA ahorran mucho tiempo.

- Shell: **zsh** con [Oh My Zsh](https://ohmyz.sh) + prompt [Starship](https://starship.rs)
- Plugins clave: `zsh-autosuggestions`, `zsh-syntax-highlighting`

## Software

### Desarrollo

- **[Bun](https://bun.sh)** — instalaciones y scripts más rápidos que npm/yarn. No siempre es posible en proyectos de equipo, pero excelente para trabajo personal
- **[Docker Desktop](https://docker.com)** — bases de datos y servicios en contenedores para mantener limpia la máquina
- **[TablePlus](https://tableplus.com)** — la mejor GUI para PostgreSQL, MySQL, SQLite y Redis
- **[Bruno](https://usebruno.com)** — cliente de API open-source, los archivos viven en el repositorio
- **[Proxyman](https://proxyman.io)** — proxy HTTP para depurar tráfico de red

### Productividad

- **[Raycast](https://raycast.com)** — lanzador, historial de portapapeles, fragmentos y gestión de ventanas. Reemplazó a Alfred y a varias apps de la barra de menú
- **[Arc](https://arc.net)** — mi navegador diario. Los Spaces y las pestañas como notas cambiaron mi forma de navegar
- **[Notion](https://notion.so)** — base de conocimiento personal, notas de proyectos y tareas
- **[CleanMyMac X](https://cleanmymac.com)** — mantiene el uso del disco bajo control

### Diseño

- **[Figma](https://figma.com)** — diseño de UI y prototipado
